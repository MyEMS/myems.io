# Use Online Demo

## Project Overview

Simulates various business formats in a low-carbon city, with a total building area of 1 million square meters, 179 meters, 284 data points, and 18 various business format subsystems, deployed on Alibaba Cloud Container Service Kubernetes cluster, with on-site installation of data collectors for data collection and upload.
